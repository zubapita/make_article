# Plan

- **プロジェクト名**: make_article
- **種別**: 実験用PoC
- **一行説明**: テーマを与えると、リサーチを行い、掲載メディア別に記事を作成する
- **作成日**: 2026-01-31

## 目的

<!-- このプロジェクトで何を実現したいか。1〜3文で書く -->
テーマや疑問点などを与えると、ユーザーと議論を行いながらWebをリサーチ。
議論やリサーチの結果をMarkdown形式でファイル保存。
それを元に構成案を作成して、さらに議論。
構成案が決定したら、記事原稿を作成してさらに議論して、内容を詰める。

## 検証したいこと

<!-- この実験で何を確かめたいか。仮説を書く -->
- APIを使わず、Claude Code、Codex CLI、Gemini CLI、OpenCodeなどをヘッドレスで運用してAIエージェント構築できるか。
- AIエージェントのSKILLはClaude Code用の標準化されたSkillを作成、保存し、それが使われるようにする。
- 複数のCLI型AIエージェントのラッパーを作成できるか。

- 仮説:CLI型AIエージェントに適切なMCPやSKILLが設定されていれば、それを呼び出すUI系のラッパーを作れば欲しいものは作れる。
